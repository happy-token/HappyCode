# HappyCode Promotional Content

## Project Overview
HappyCode is a desktop GUI for Claude Code built with Electron + React + TypeScript. It provides a professional, enterprise-ready interface for AI-powered coding.

## Key Features

### Session Audit Viewer
- Full transparency: Browse and inspect all Claude Code sessions.
- Detailed insights: View tool calls, inputs, outputs, costs, and token usage.
- Compliance ready: Export session history to CSV for offline records.

### Advanced Chat UI
- Modern Interface: Syntax-highlighted code blocks with one-click copy.
- Real-time Monitoring: Live cost display per conversation.
- Smart Permissions: Interactive dialogs for tool approval.
- Model Switching: Pick your model between turns.
- Multi-Agent Visualization: Subagent tree for complex sessions.

### Future Roadmap
- Hooks Configuration UI
- CLAUDE.md Editor
- Multi-session Tabs
- PDF Export
- Team Audit Server

## Tech Stack
- Electron 31+
- React 19 + TypeScript
- Tailwind CSS / shadcn/ui
- @anthropic-ai/claude-agent-sdk

## Value Proposition
Empower your enterprise developers with the power of Claude Code through a secure, auditable, and beautiful desktop interface.
